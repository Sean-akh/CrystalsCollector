CRYSTAL COLLECTOR

Rules of the Game

  • Below there are four crystals as botton
  • A random number is displayed at the start of the game under "the Game Score".
  • When click on a crystal, it will add a specific amount of points to the player's total score.
  • The value of each Crystal is hidden. But the overall score is updated
  • The game is won when the player's score match the Game's score. The player loose if the score is more than the Game's score.
